# Инвентаризация исходных документов базы знаний электриков

Дата создания: 2026-06-16

## Назначение

Этот файл фиксирует исходные документы, из которых будет собираться база знаний для электриков, монтажников, менеджеров и инженеров "Светон".

Статусы:

- `raw`: исходный файл перенесен в постоянное хранилище;
- `text_pending`: текст еще не извлечен;
- `extracted`: текст извлечен;
- `chunked`: документ разбит на чанки;
- `statements_extracted`: извлечены атомарные утверждения;
- `review_required`: требуется ручная проверка;
- `approved`: материал прошел проверку и может использоваться в базе знаний.

Примечание по текущему извлечению:

- `DOCX -> Markdown`: качество извлечения рабочее, можно использовать для следующего шага.
- `PDF -> txtwrite (ghostscript)`: текст извлечен, но у PDF заметны артефакты layout. Перед semantic extraction требуется ручная проверка или улучшенный extractor.
