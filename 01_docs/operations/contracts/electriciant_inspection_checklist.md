# Чек-лист осмотра места монтажа ИБП

Назначение: самопроверка электрика для южного сценария. Электрик заполняет чек-лист на объекте и отправляет его в Светон вместе с заявкой на подбор оборудования, отчетом и фотоотчетом.

## 1. До начала осмотра

- [ ] Разместить термометр в предполагаемом месте установки АКБ.
- [ ] Уточнить у клиента, какие помещения, щиты и трассы можно фотографировать.

## 2. Щит и электросеть

- [ ] Фото открытого щита «на столбе».
- [ ] Фото основного щита до снятия защитной крышки.
- [ ] Фото списка потребителей, если прилагается к щиту.
- [ ] Снять защитную крышку щита.
- [ ] Фото открытого щита полностью.
- [ ] Фото щита и номиналы каждого ряда со всех сторон (подробно, много фото).
- [ ] Проверить PE/N шины.
- [ ] Измерить напряжение на каждой фазе.
- [ ] Установить соответствие автоматов и нагрузок, планируемых в резерв.
- [ ] Заполнить в отчете информацию по всем автоматам и нагрузкам из резервной группы.
- [ ] Проверить наличие мастер выключателя, проверить, какие потребители он обеспечивает, сделать фото.
- [ ] Фото показаний термометра, записать эти показания.
- [ ] Проверить наличие дополнительных щитов, которые относятся к работам по выделению резервной группы.

## 3. Потребители и оборудование

- [ ] Померить номинальную и пиковую мощность погружного насоса (или другого оборудования, для которого нет данных).
- [ ] Фото шильдиков циркуляционных насосов и другого резервируемого оборудования по возможмости.
- [ ] Проверить, все ли заявленные потребители установлены фактически.
- [ ] Отметить в отчете  все спорные или непонятные нагрузки.
- [ ] Посчитать итоговую мощность всех потребителей из резерва.

## 4. Место установки и трасса

- [ ] Сфотографировать место установки ИБП/инвертора/АКБ по кругу.
- [ ] Записать размеры помещения, если это нужно для монтажа.
- [ ] Осмотреть предполагаемую трассу кабеля.
- [ ] Фото трассы на всем протяжении.
- [ ] Фото места проходов через стены/перекрытия.

## 5. Обязательно проверить

- [ ] Проверить наличие газового отсекателя для газового котла.
- [ ] Уточнить у клиента, есть ли трехфазные нагрузки.
- [ ] Проверить, нет ли радиатора отопления рядом с местом установки ИБП. 
- [ ] Уточнить пирог/толщину стены, если будут проходы.

## 6. Заявка на подбор оборудования

- [ ] Заполнить заявку на подбор оборудования.
- [ ] Подписать заявку у клиента 
- [ ] Сфотографировать/отсканировать подписанную заявку.

## 7. Материалы для отправки в Светон

- [ ] Фото/скан подписанной заявки на подбор оборудования.
- [ ] Полностью заполненный отчет.
- [ ] Этот заполненный чек-лист.
- [ ] Скриншот координат объекта.
- [ ] Фото отчет

## 8. Завершение

- [ ] Проверил, что все действия по чек-листу выполнены или отмечены как невыполнимые.
- [ ] В отчете указал, что не удалось проверить и почему.
- [ ] Передал комплект материалов в Светон.
