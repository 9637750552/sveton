# Проект "Светон-Юг"

Проект по расширению бизнеса ИБП (Sveton / Bidneos) в Краснодарский край и прилегающие регионы.

## Структура проекта

### 1. [input/](./input/) — База знаний и исходные данные
Сырье для создания стратегии и контента:
- `interviews/`: Тексты интервью с ГД и КД.
- `documents/`: Внешние PDF и презентации (УТП).
- `catalog/`: Тех. характеристики с основного сайта.
- `portfolio/`: Примеры работ в Москве.
- `legal/`: Юридические данные и политики.

### 2. [docs/](./docs/) — Управляющие документы
Стратегия и регламенты:
- `governance/`: SSOT (PROJECT_STATUS), GUIDE (стандарты), DEVELOPMENT_LOG.
- `strategy/`: EXECUTIVE_SUMMARY, SWOT, ROADMAP.
- `operations/`: Рабочие чек-листы.

### 3. [results/](./results/) — Результаты работы
Артефакты, полученные в ходе выполнения плана:
- `economics/`: Расчеты логистики и прайс-листы.
- `market/`: Анализ конкурентов и базы контактов.
- `legal/`: Шаблоны договоров для Юга.
- `marketing/`: Офферы и рекламные материалы.
- `operations/`: Отчеты об аудитах и кейсы.

### 4. [website/](./website/) — Разработка сайта
Площадка для сборки нового сайта "Светон-Юг".
