# Чек-лист электрика по осмотру объекта

Назначение: самопроверка электрика для южного сценария. Электрик заполняет чек-лист на объекте или сразу после осмотра и отправляет его в Светон вместе с клиентской анкетой, отчетом для ТЗ и фотоотчетом.

## 1. До начала осмотра

- [ ] Представился клиенту как электрик, выполняющий осмотр для Светон.
- [ ] Получил доступ к щиту, месту установки и предполагаемой трассе.
- [ ] Разместил термометр в предполагаемом месте установки АКБ.
- [ ] Уточнил у клиента, какие помещения, щиты и трассы можно фотографировать.

## 2. Клиентская анкета

- [ ] Заполнил с клиентом клиентскую анкету.
- [ ] Проверил, что в анкете заполнены ключевые поля: нагрузки, ввод, условия установки, вводные автоматы, особенности.
- [ ] Получил подпись клиента в анкете.
- [ ] Сфотографировал/отсканировал подписанную анкету.

## 3. Щит и электросеть

- [ ] Сфотографировал щит до открытия.
- [ ] Безопасно открыл щит, если это возможно и допустимо.
- [ ] Сфотографировал открытый щит полностью.
- [ ] Сфотографировал каждый ряд автоматов так, чтобы были видны номиналы.
- [ ] Сфотографировал вводной автомат.
- [ ] Сфотографировал PE/N шины.
- [ ] Измерил напряжение по фазам и внес значения в клиентскую анкету.
- [ ] Проверил соответствие автоматов и потребителей, планируемых в резерв.
- [ ] Заполнил в отчете для ТЗ раскладку всех автоматов по каждому щиту, где возможны работы; детали указал для резервных и спорных линий.
- [ ] Если автоматов больше, чем строк в таблице, зарисовал щит вручную и приложил фото схемы.
- [ ] Проверил, есть ли трехфазные потребители в резерве.
- [ ] Проверил, есть ли мастер-выключатель и что он отключает.
- [ ] Проверил наличие дополнительных щитов, которые относятся к работам.

## 4. Потребители и оборудование

- [ ] Проверил мощность насоса/насосов и другого оборудования, если данных нет в анкете.
- [ ] Сфотографировал шильдики насосов и важного оборудования.
- [ ] Проверил количество циркуляционных насосов и иных обязательных потребителей.
- [ ] Проверил, все ли заявленные потребители установлены фактически.
- [ ] Отметил в отчете для ТЗ все спорные или непонятные нагрузки.

## 5. Место установки и трасса

- [ ] Сфотографировал место установки ИБП/инвертора/АКБ по кругу.
- [ ] Сфотографировал расположение щита относительно места установки.
- [ ] Записал размеры помещения, если это нужно для монтажа.
- [ ] Через 40 минут снял показания термометра и внес их в анкету или отчет.
- [ ] Проверил, нет ли радиатора отопления рядом с местом установки.
- [ ] Проверил, нет ли газового отсекателя котла рядом с местом установки.
- [ ] Осмотрел предполагаемую трассу кабеля.
- [ ] Сфотографировал трассу на всем протяжении.
- [ ] Сфотографировал места проходов через стены/перекрытия.
- [ ] Уточнил пирог/толщину стены, если будут проходы.

## 6. Материалы для отправки в Светон

- [ ] Фото/скан подписанной клиентской анкеты.
- [ ] Отчет электрика после осмотра для подготовки ТЗ.
- [ ] Этот заполненный чек-лист.
- [ ] Скриншот координат объекта.
- [ ] Фото щита до открытия.
- [ ] Фото открытого щита.
- [ ] Фото вводного автомата и рядов автоматов.
- [ ] Фото PE/N шин.
- [ ] Фото места установки по кругу.
- [ ] Фото трассы и сложных мест.
- [ ] Фото шильдиков насосов/оборудования, если есть.

## 7. Завершение

- [ ] Проверил, что все действия по чек-листу выполнены или отмечены как невыполнимые.
- [ ] В отчете для ТЗ указал, что не удалось проверить и почему.
- [ ] Передал комплект материалов в Светон.
