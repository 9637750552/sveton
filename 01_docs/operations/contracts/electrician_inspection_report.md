## 2. Раскладка автоматов по щитам

Перечислить все автоматы по порядку в каждом щите, где возможны работы. Если щитов несколько, в колонке `Щит / место` указывать понятное название: `щит дома`, `щит котельной`, `уличный щит`, `щит гаража` и т.п.

Для нерезервных автоматов достаточно заполнить `N`, `Щит / место`, `Ряд / позиция`, `Номинал`, `Подпись на щите` и `Резервный?`. Колонки `Что фактически питает`, `УЗО/диф`, `Фаза`, `Комментарий для ТЗ` обязательно заполняются только для резервных автоматов и спорных линий.
