# Необходимые ОКВЭД для ИП (Проект Светон-Юг)

На основании аудита выписки ЕГРИП от 24.05.2026, для легальной работы филиала необходимо добавить следующие коды деятельности:

1. **43.21** — Электромонтажные работы (Монтаж систем ИБП, пусконаладка).
2. **47.54** — Торговля розничная бытовыми электротоварами (Продажа ИБП физлицам).
3. **46.69.5** — Торговля оптовая производственным электротехническим оборудованием (Продажа комплектов B2B).
4. **46.18.9** — Деятельность агентов по оптовой торговле прочими товарами (Получение комиссии от ГО Светон).

*Статус:* Необходимо подать заявление в ФНС.
