## 4. Ожидаемые Statement Groups

Рекомендуемые смысловые группы для будущего extraction:

1. `work_on_site.scope_and_sequence`
   - Работа на объекте выполняется в паре.
   - Стандартный порядок может меняться на усмотрение первого номера, если место на объекте ограничивает действия.
   - При сложных задачах основной процесс может выполняться совместно или самостоятельно через самораспределение.

2. `work_on_site.first_number_responsibility`
   - Первый номер осматривает место монтажа, согласует изменения с клиентом, контролирует ключевые действия.
   - Первый номер отвечает за решение, какие действия может выполнить стажер или второй номер на месте.
   - Сотрудник со 2 разрядом отвечает за выполнение работ на выезде и решение вопросов с клиентом.

3. `work_on_site.trainee_limits`
   - Стажер первые выезды наблюдает за работой первого номера.
   - Стажер может собирать стеллаж, готовить провода, собирать группу АКБ, изготавливать провода балансира, собирать DC-защиту и прокладывать трассу при наличии.
   - Концы проводов, которые подключаются к инвертору, должны быть заизолированы.
   - Стажер ни при каких условиях не подключает основные провода по постоянному току.

4. `work_on_site.battery_and_dc_supervision`
   - Первый номер первые несколько раз подключает провода балансира вместе со стажером.
   - Первый номер проверяет корректность сборки группы АКБ и учит стажера проверке АКБ и балансира.
   - В дальнейшем первый номер только контролирует работу стажера на этом этапе.
   - Подключение группы АКБ к инвертору и подключение балансира выполняется первым номером, когда второй участник стажер.

5. `work_on_site.first_second_number_distribution`
   - При паре "1 разряд / второй номер" первый номер ведет инвертор, байпасный щит, трассу, проход в щит клиента и переборку щита.
   - Второй номер может самостоятельно собирать группу АКБ, готовить провода, подключать балансир, подключать основные DC-провода к инвертору, протягивать клеммные болты, ставить защитные панели и прокладывать трассу при наличии.
   - Действия второго номера могут меняться в зависимости от навыков.
   - Большую трассу могут делать совместно.

6. `work_on_site.handover_and_closeout`
   - Сдача оборудования клиенту включает проверку резервной группы, имитацию отключения электричества, консультацию по байпасному щиту, уборку, сбор инструмента, прием оплаты, подпись документов и написание отчета.
   - Для пары со 2 разрядом дополнительно встречается настройка умной/GSM-розетки и подготовка документации.

7. `work_on_site.reporting_and_media`
   - Настройка инвертора сопровождается фотографированием результатов и видеозаписью для отчета.
   - Выполнение этого этапа может быть по договоренности первым или вторым номером.
