# Анкета первичного отбора электриков-монтажников

Формат: Яндекс Форма или аналогичная анкета.

Цель: не обзванивать всех кандидатов подряд. Сначала собрать структурированные ответы, автоматически отсечь слабые отклики и созваниваться только с теми, кто проходит базовый фильтр.

Принцип формы: по возможности использовать только выбор одного или нескольких вариантов. Свободный текст оставить только там, где без него нельзя.

Рекомендуемая воронка:

1. Отправить анкету кандидатам после первичного отклика на объявление.
2. Дать 24-48 часов на заполнение.
3. Автоматически отсечь ответы со стоп-факторами.
4. Отсортировать оставшихся по баллам.
5. Созвониться только с верхней группой кандидатов.
6. После звонка дать тест по фото щита или пилотный простой объект.

## 1. Сообщение кандидату

Здравствуйте. Спасибо за отклик.

Чтобы не тратить ваше и наше время на длинный первый звонок, просим заполнить короткую анкету. Мы ищем электриков под монтаж резервного питания и ИБП на объектах клиентов в Краснодарском крае. Работа идет по техническому заданию, с фотоотчетом и удаленным инженерным контролем.

Анкета занимает 5-7 минут: [ссылка]

После ответов выберем кандидатов для созвона.

## 3. Стоп-факторы

Кандидата не звать на интервью, если выполнен хотя бы один пункт:

- в вопросе 3 выбран вариант `Не Краснодарский край`;
- в вопросе 8 выбран вариант `С щитами почти не работаю`;
- в вопросе 9 выбран вариант `Ничего из перечисленного`;
- в вопросе 10 выбран вариант `Нет`;
- в вопросе 12 выбран вариант `Пока не готов`.

## 4. Балльная модель

Баллы можно назначать автоматически по ответам:

| Критерий | Баллы |
|---|---:|
| Вопрос 5: `В день обращения` | 2 |
| Вопрос 5: `На следующий день` или `В течение 2-3 дней` | 1 |
| Вопрос 6: `3-5 лет` | 2 |
| Вопрос 6: `Более 5 лет` | 3 |
| Вопрос 7: выбран `Частные дома` | 2 |
| Вопрос 7: выбран `Коммерческие помещения` | 1 |
| Вопрос 8: `Регулярно собираю и дорабатываю щиты` | 3 |
| Вопрос 8: `Иногда собираю или дорабатываю` | 1 |
| Вопрос 9: выбран `ИБП` | 2 |
| Вопрос 9: выбран `Генераторы` или `Стабилизаторы` или `АВР / ручной ввод резерва` | 1 |
| Вопрос 10: `Да, приложу сразу к анкете` | 2 |
| Вопрос 10: `Да, отправлю в мессенджер после анкеты` | 2 |
| Вопрос 10: `Смогу прислать позже по запросу` | 1 |
| Вопрос 11: `Бесплатно при дальнейшем монтаже` | 2 |
| Вопрос 11: `До 2 000 руб.` | 1 |
| Вопрос 12: `Да, готов брать объекты в ближайшее время` | 2 |
| Вопрос 12: `Частично, зависит от даты` | 1 |

Максимум: `21` балл.

Интерпретация:

- `15-21` - звать на интервью в первую очередь;
- `10-14` - рабочий резерв, можно звать при нехватке сильных кандидатов;
- `0-9` - не тратить время на звонок, кроме редкой географии или сильных примеров работ.

## 5. Практическая схема отбора

1. Отправить анкету всем релевантным откликнувшимся.
2. Через 24 часа напомнить тем, кто не ответил.
3. Через 48 часов закрыть первый сбор.
4. Отфильтровать ответы со стоп-факторами.
5. Оставшихся отсортировать по баллам.
6. Взять верхнюю группу на короткий созвон по [ELECTRICIAN_INTERVIEW_SCRIPT.md](ELECTRICIAN_INTERVIEW_SCRIPT.md).
7. После созвона выбрать 3-5 кандидатов на тест по фото щита или на первый простой пилот.

Важно: эта версия анкеты сама по себе не проверяет техническую логику, дисциплину фотоотчета и работу по ТЗ. Это переносится на этап созвона.

- готовность прислать фото прошлых работ;
- стоимость выезда;
- доступность по времени;
- итоговые баллы;
- стоп-фактор `да / нет`;
- комментарий менеджера после проверки.

## 7. Практический смысл анкеты

Эта анкета должна отсеивать не по "красоте ответов", а по рабочей пригодности под процесс Светон:

- локальность и реальную готовность к выезду по Краснодарскому краю;
- наличие базового релевантного опыта по щитам и смежному оборудованию;
- готовность показать реальные примеры своих работ;
- разумную стоимость первого выезда;
- реальную доступность по срокам;
- пригодность к следующему этапу, где уже проверяются ТЗ, фотоотчет, удаленная приемка и техническая логика;
- партнерскую модель работы, а не ожидание штатного трудоустройства.
