# База знаний Sveton (Архив и Вводные данные)

В этой папке собрана вся исходная информация для проекта "Светон-Юг".

## Структура папок:
- `interviews/`: Тексты интервью с Генеральным и Коммерческим директорами.
- `documents/`: Внешние файлы (Презентации, PDF с точками продаж).
- `catalog/`: Технические характеристики и оборудование (с основного сайта).
- `portfolio/`: Кейсы и примеры работ МСК.
- `legal/`: Юридические политики и данные компании.
- `articles/`: Маркетинговые статьи.

## Использование:
Данные из этой папки используются как "сырье" для создания стратегии, контента нового сайта и операционных чек-листов.
