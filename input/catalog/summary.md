# Сводный каталог систем (с сайта sveton-ibp.ru)

## Системы 3 кВт (Lite/Standard)
- **Базовый 3кВт / 2кВтч:** ~106к руб. (Резерв 6-10 ч).
- **Базовый 3кВт / 4кВтч:** ~152к руб. (Резерв 12-20 ч).
- **Контроль 3кВт / 8кВтч Компакт:** ~299к руб. (Резерв 20-36 ч).

## Системы 5 кВт (Standard)
- **Базовый 5кВт / 4кВтч:** ~182к руб. (Резерв 5-9 ч).
- **Базовый 5кВт / 8кВтч:** ~283к руб. (Резерв 10-18 ч).
- **Контроль 5кВт / 10кВтч LiFe-W:** На LiFePO4 аккумуляторах.
- **Контроль 5кВт / 16кВтч:** ~398к руб. (Резерв 20-36 ч).

## Системы 10-15 кВт и более (Professional)
- **Контроль 10кВт / 8кВтч серия R:** ~398к руб.
- **Контроль 10кВт / 32кВтч серия R:** ~1.06 млн руб. (Резерв 20-48 ч).
- **Контроль 15кВт / 16кВтч серия R:** Для больших коттеджей.
- **Мощные системы:** До 45 кВт (индивидуальный расчет).

## Технические требования для подбора (Deal Killers):
- Необходим **проходной ноль** и **чистый синус** для котлов.
- Важно разделение групп в щитке (резервируемая группа).
- Учет пусковых токов (насосы, холодильники).
