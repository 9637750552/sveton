# Проект "Светон-Юг"

Проект по расширению бизнеса ИБП (Sveton / Bidneos) в Краснодарский край и прилегающие регионы.

## Структура проекта

### 1. [input/](./input/) — База знаний и исходные данные
Сырье для создания стратегии и контента:
- `briefs/`: Короткие вводные по оборудованию, маркетингу и техническим деталям.
- `interviews/`: Тексты интервью с ГД и КД.
- `documents/`: Внешние PDF и презентации (УТП).
- `catalog/`: Тех. характеристики с основного сайта.
- `portfolio/`: Примеры работ в Москве.
- `legal/`: Юридические данные и политики.
- `assets/`: Исходные изображения и списки URL для медиаматериалов.

### 2. [docs/](./docs/) — Управляющие документы
Стратегия и регламенты:
- `governance/`: SSOT (PROJECT_STATUS), GUIDE (стандарты), DEVELOPMENT_LOG.
- `strategy/`: EXECUTIVE_SUMMARY, SWOT, ROADMAP.
- `operations/`: Рабочие чек-листы.
- `reference/`: Справочные объяснения и материалы, которые не являются планами или результатами.

### 3. [results/](./results/) — Результаты работы
Артефакты, полученные в ходе выполнения плана:
- `economics/`: Расчеты логистики и прайс-листы.
- `legal/`: Шаблоны договоров для Юга.
- `marketing/`: Офферы и рекламные материалы.
- `operations/`: Отчеты об аудитах и кейсы.

### 4. [images/](./images/) — Готовые визуальные материалы
Медиа, которые можно использовать в документах, объявлениях и сайте:
- `banners/`: Готовые баннеры.
- `generated/`: Сгенерированные изображения и варианты.
- `logo/`: Логотипы и иконки.

### 5. [research/](./research/) — Исследования
Гипотезы, география, анализ рынка и черновики исследовательской базы.

### 6. [website/](./website/) — Разработка сайта
Площадка для сборки нового сайта "Светон-Юг".

### 7. [scripts/](./scripts/) — Служебные скрипты
Конвертация Markdown в DOCX/PDF и другие автоматизации.

### 8. [tmp/](./tmp/) — Локальные временные файлы
Папка игнорируется git. Используется для превью, промежуточных экспортов и мусора от офисных программ.
