# Маркетинговые тезисы и "боли" (для сайта Юга)

## Почему ИБП лучше генератора (УТП):
1. **Тишина и комфорт:** Нет шума, запаха выхлопных газов. Можно ставить прямо в доме.
2. **Мгновенная реакция:** 0 секунд на переключение. Электроника (котлы, ПК) не перезагружается.
3. **Автономность:** Работает без участия человека (не надо заправлять топливом).
4. **Надежность:** 100% запуск всегда. Не зависит от качества бензина/дизеля.
5. **Экономия:** Не требует дорогого регулярного ТО.

## Целевые группы и их боли:
- **Частники:** Разморозка дома (котел), отсутствие воды (насос), Wi-Fi.
- **Бизнес:** Остановка касс, потеря клиентов, порча продуктов (холодильники).
- **Специфика Юга:** Огромный износ сетей, жара, выход из строя кондиционеров.

## Стандарты сервиса Светон:
- 17 лет на рынке (экспертиза).
- Бесплатный выезд на осмотр (в МСК).
- Фиксированная стоимость после осмотра.
- Гарантия на систему целиком.
