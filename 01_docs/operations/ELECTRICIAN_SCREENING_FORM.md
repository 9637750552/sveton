# Анкета первичного отбора электриков-монтажников

Формат: Яндекс Форма или аналогичная анкета.

Цель: не обзванивать всех кандидатов. Сначала собрать структурированные ответы, автоматически отсеять слабых и созвониться только с теми, кто прошел базовый фильтр.

Рекомендуемая воронка для 76 кандидатов:

1. Отправить всем короткое сообщение со ссылкой на анкету.
2. Дать 24-48 часов на заполнение.
3. Отсечь неподходящих по стоп-факторам.
4. Отобрать 10-15 лучших по баллам.
5. С ними провести короткое интервью по `ELECTRICIAN_INTERVIEW_SCRIPT.md`.
6. 3-5 лучших вывести на тестовое задание по фото щита.

## 1. Сообщение кандидату

Здравствуйте. Мы запускаем направление по резервному питанию для частных домов: ИБП, аккумуляторы, байпас, выделение критичных нагрузок.

Ищем электриков-монтажников в Краснодарском крае для работ на объектах под удаленным контролем инженера. Чтобы не тратить ваше и наше время на длинный первый звонок, просим заполнить короткую анкету. По результатам выберем мастеров для созвона и первых объектов.

## 3. Стоп-факторы

Кандидата не звать на интервью, если есть один или несколько признаков:

- не готов работать по схеме;
- не готов присылать фото до включения;
- не готов ждать удаленной проверки перед запуском;
- не собирает щиты и не работал с частными домами;
- пишет ответы в стиле "я сам все знаю, мне схемы не нужны";
- не понимает, зачем нужен байпас;
- предлагает резервировать весь дом без расчета нагрузок;
- нет инструмента для нормальной работы со щитом.

Максимум: 28 баллов.

Интерпретация:

- 22-28: звать на интервью в первую очередь.
- 16-21: резерв, можно звать при нехватке сильных кандидатов.
- 0-15: не тратить время на звонок, кроме случаев редкой географии.

## 5. Практическая схема отбора из 76 человек

1. Отправить анкету всем 76.
2. Через 24 часа напомнить тем, кто не ответил.
3. Через 48 часов закрыть первый сбор.
4. Отсеять всех со стоп-факторами.
5. Отсортировать оставшихся по баллам.
6. Взять топ-10 на короткий созвон.
7. После созвона выбрать 3-5 на тест по фото щита.
8. Одного-двух оставить кандидатами на первый пилотный монтаж.
