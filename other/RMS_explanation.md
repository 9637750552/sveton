# Среднеквадратичное напряжение (RMS) — подробное объяснение

## Зачем нужен RMS

В сети 230 В напряжение не постоянно. Оно меняется по синусоиде от примерно +325 В до -325 В.

Если взять обычное среднее напряжение за период, получится 0 В, потому что положительная и отрицательная полуволны взаимно компенсируются.

Но энергия при этом передаётся, лампы светят и чайники работают. Поэтому нужен другой способ измерения «силы» переменного напряжения.

## Почему появляется квадрат

Для резистора мощность определяется формулой:

P = U² / R

То есть нагрев зависит от квадрата напряжения.

+100 В и -100 В дают одинаковый нагрев, потому что:

100² = 10000
(-100)² = 10000

Поэтому для оценки энергетического воздействия сигнала нужно усреднять квадрат напряжения.

## Определение RMS

Алгоритм:

1. Берём напряжение в каждый момент времени.
2. Возводим его в квадрат.
3. Усредняем за период.
4. Извлекаем квадратный корень.

Отсюда название: средне-квадратичное значение.

## Формула RMS

U_RMS = sqrt((1/T) * ∫ u²(t) dt)

где:
- u(t) — напряжение в момент времени t
- T — период сигнала

## Синусоида сети

u(t) = U_peak × sin(2πft)

где:
- U_peak — амплитуда (пиковое значение)
- f — частота в герцах
- t — время

Для сети:
- f = 50 Гц
- U_RMS = 230 В
- U_peak ≈ 325 В

## Что такое радиан

Радиан — единица измерения угла.

Соотношения:
- π радиан = 180°
- 2π радиан = 360°
- π/2 радиан = 90°

## Откуда появляется корень из двух

Для синусоиды среднее значение sin² за полный период равно 1/2.

Поэтому:

Средний квадрат = U_peak² / 2

После извлечения корня:

U_RMS = U_peak / √2

или

U_peak = U_RMS × √2

## Пример для сети

230 × 1.414 ≈ 325 В

То есть розетка 230 В RMS имеет пики около +325 В и -325 В.

## Разные способы усреднения

Для синусоиды с амплитудой 325 В:

- обычное среднее = 0 В
- среднее по модулю ≈ 207 В
- RMS = 230 В
- пик = 325 В

Все значения правильные, но описывают разные свойства сигнала.

## Вращающийся вектор

Синусоиду удобно представлять как проекцию вращающегося вектора.

Мгновенное напряжение — это проекция вектора на ось.

RMS — это не средняя проекция.

RMS — это корень из среднего квадрата этой проекции за полный оборот.

## Практический смысл

Когда на оборудовании написано 230 В, почти всегда имеется в виду 230 В RMS.

Именно это значение используют:
- мультиметры
- электрики
- производители оборудования
- расчёты мощности
